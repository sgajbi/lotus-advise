# RFC Archive

This directory contains RFC material preserved for historical traceability, not active implementation planning.

Current archive sets:

| Archive Set | Reason |
| --- | --- |
| `raw-advisory-drafts/` | Early advisory draft RFCs superseded by the flat active RFC sequence in `docs/rfcs/`. |

Archive policy:
- Archived RFCs are not the source of truth for current planning.
- Active references should point to the current top-level RFCs under `docs/rfcs/`.
- Archived material may still be useful for historical rationale or migration context.
