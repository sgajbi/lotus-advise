# Getting Started

## Install

```bash
make install
```

## Run locally

```bash
make run
```

Then open `/docs` or probe:

- `/health`
- `/health/live`
- `/health/ready`

## Fast local proof

```bash
make check
```

## Canonical local Docker upstream URLs

- `http://core-control.dev.lotus`
- `http://core-query.dev.lotus`
- `http://risk.dev.lotus`

These remain part of the repo’s documented local runtime contract.
