# Lotus Advise Third-Party Notices

Lotus Advise depends on third-party Python packages declared in `requirements-prod.txt` and
`requirements-dev.txt`. The governed release inventory is generated by `make license-ip-inventory`
and committed at `docs/standards/license-ip-inventory.v1.json`.

The license/IP gate classifies runtime and development dependency graphs through
`docs/standards/license-ip-policy.v1.json`. Any review-required license must have an explicit,
owned, expiring exception before release evidence can pass.

Generated advisory content, demo evidence, API inventories, quality reports, SBOMs, vulnerability
scan reports, signatures, and provenance attestations are Lotus-authored release artifacts unless
their source files explicitly identify a third-party origin.
