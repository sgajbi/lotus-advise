# Roadmap

## Current phase

`lotus-advise` is in an advisory-workflow hardening phase with explicit runtime and lifecycle
governance.

## Delivered foundations

- advisory-only workflow ownership after the split
- stateful and stateless proposal simulation
- proposal lifecycle persistence and workflow state
- workspace drafting and lifecycle handoff
- persisted decision summary and proposal alternatives

## Intentional limits

- management workflows remain outside this repo
- performance analytics are not currently an advisory input contract
- gRPC is not justified for current upstream advisory calls
